# Cloud DevOps Lab

**AWS-focused cloud and DevOps lab with practical projects, infrastructure as code, and automation workflows.**

---

## About

This repository is my personal knowledge base and hands-on lab where I design, build, and document cloud solutions using AWS and modern DevOps practices.

The goal is not just to learn concepts, but to apply them in real-world scenarios:
- Designing architectures  
- Deploying infrastructure  
- Automating workflows  
- Solving practical problems  

---

## What you'll find here

### AWS & Cloud
- Core services (Lambda, API Gateway, DynamoDB, S3, etc.)
- Architecture patterns and design decisions
- Security and IAM practices

### DevOps & Automation
- Infrastructure as Code (Terraform)
- CI/CD pipelines (GitHub Actions)
- Deployment strategies

### Containers
- Docker fundamentals
- Containerizing applications
- Running services in cloud environments

### Projects
Real-world inspired implementations combining multiple tools:

- Serverless APIs (Lambda + API Gateway + DynamoDB)
- Containerized applications (Docker + ECS)
- Infrastructure provisioning with Terraform
- End-to-end CI/CD pipelines

Each project includes:
- Architecture diagrams  
- Implementation steps  
- Key decisions and trade-offs  
- Lessons learned
